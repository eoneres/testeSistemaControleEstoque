# Sistema de Gerenciamento de Estoque

Sistema completo para gerenciamento de estoque desenvolvido em Python com Tkinter e SQLite.

## 📋 Funcionalidades

- ✅ Login de usuários
- ✅ Cadastro de produtos
- ✅ Consulta de estoque com busca
- ✅ Registro de vendas
- ✅ Relatórios diversos
- ✅ Gerenciamento completo de produtos
- ✅ Movimentação de estoque
- ✅ Histórico de movimentações
- ✅ Código de barras

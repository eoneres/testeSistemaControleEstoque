# Sistema de Gerenciamento de Estoque

**Projeto acadêmico desenvolvido para a disciplina de Programação**

## Sobre o Projeto

Este sistema foi desenvolvido como trabalho prático da disciplina de Desenvolvimento de Sistemas, com o objetivo de criar uma solução completa para gerenciamento de estoque para pequenas empresas.

**Período:** 3º semestre de Análise e Desenvolvimento de Sistemas  
**Disciplina:** Programação Desktop  
**Tecnologias:** Python, Tkinter, SQLite

## Funcionalidades Implementadas

- **Módulo de Login** - Controle de acesso de usuários
- **Cadastro de Produtos** - Interface intuitiva para registro de itens
- **Consulta de Estoque** - Busca dinâmica com destaque para itens com estoque baixo
- **Registro de Vendas** - Baixa automática no estoque e cálculo de totais
- **Relatórios Gerenciais** - Estoque atual, produtos críticos e vendas do dia
- **Movimentação de Estoque** - Controle de entradas, saídas e ajustes
- **Histórico Completo** - Rastreabilidade de todas as alterações

## Tecnologias Utilizadas
